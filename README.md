# Sistema Integral de Gestión de Clientes, Servicios y Reservas

Sistema desarrollado en Python para gestionar clientes, servicios y reservas con manejo de errores y logging.

## Tecnologías
- Python 3.14
- Programación Orientada a Objetos
- Manejo de excepciones personalizadas
- Logging

## Funcionalidades
- Crear, listar, actualizar y eliminar clientes
- Crear, listar, actualizar y eliminar servicios  
- Crear reservas validando cliente y servicio
- Manejo de errores con try/except
- Registro de operaciones en `sistema.log`
